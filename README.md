## Installation
- Clone the repository using `git clone` and then change the directory to root of the project
```
    https://github.com/chandrdubey/Jumbo-youtube-task.git
    cd Jumbo-youtube-task
```

```bash
$ npm install
```

## Migration

```bash
$ npm run migrate:up
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
